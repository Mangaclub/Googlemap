Functions Version 4.2.3

Google Map Karten erstellen und anzeigen

  • Karten erstellen und bearbeiten nach der WYSIWYG Methode
  • vollständige Kopien von Karten erstellen
  • alle Google Map Kartenparameter lassen sich individuell einstellen
  • Zentraler Karten-Steuer-Button
  • Integration der Karten Inhalte (Marker Beschreibung) in die Seiten Suche
  • Frontend mit mobile device Erkennung für den Steuerbutton
  • Integration der Karten als Link in den Beiträgen und Ausgabe über ein Modales Fenster
  • Full Screen Modus
  • Responsives Map Design
  • Integration der Karten im Beitrags-Editor und Modulen
  • Sprache der Karte gleich Website-Sprache
  • de, en, fr sind installiert, eigene Sprachen können für den Steuerbutton erstellt werden
  • Linien Arten Editor

 Erweiterte Kartenoptionen

  • kleine Übersichtskarte
  • Zoombegrenzung
  • Integration von Streetview
  • Anzeige von Radwege, Straßenverkehr, Öffentliche Verkehrsmittel (z.B. U-Bahn Linien)

Erstellen von eigenen Inhalten

  • Erstellen von Markern per Autocomplete
  • Cluster Marker Funktion
  • Marker mit Mouse over Text, optional kann ein Marker gewählt werden, welcher beim Kartenstart ein geöffnetes Infofenster hat
  • Icon Auswahl für Marker per Baumanzeige
  • Erstellen von Rechtecken und Kreisen per Maus direkt auf der Karte
  • Geodäte Linien erstellen durch setzen von Markierungen auf der Karte per Mausklick. Hinzufügen und Löschen von Wegpunkten auch innerhalb der Linie möglich.
  • Höhenschnitt und Längenberechnung für Linien
  • Geodäte Polygone erstellen wie Linentool, jedoch als geschlossene Fläche
  • Umfang und Flächenberechnung für Polygone
  • Einfügen von Texten, Bildern, Links etc. (HTML Box)
  • Info Fenster jetzt für alle Elemente möglich
  • Zugriffsebenen Steuerung aller selbst erstellten Karten Elemente

Google map provide and indicate

  • Creating and editing maps by the WYSIWYG method
  • create full copies of maps
  • Google Map all parameters can be set individually
  • Central map control button
  • Integration of map content in the Site Search
  • Frontend with mobile device detection for the control button
  • Integration of the maps as a link in the articles and issue a modal window
  • Integration of the maps in the post editor and modules
  • Language map at the same site language
  • de, en, fr are installed, their own languages ​​can be created on the control button

 Extended map options

  • small general map
  • Zoom limit
  • Integration of Streetview
  • Show of cycle tracks, traffic, public means of transport (e.g. underground lines)

Provide from own contents

  • Creating markers
  • Cluster marker function
  • Marker with Mouse over text, optionally can be selected a marker, which has an opened info-window with the map start
  • Icon selection for markers by tree view
  • Creating rectangles and circles with the mouse directly on the map
  • Create by setting markers on the map by mouse click geodesic lines. Adding and deleting of waypoints within the line possible.
  • Height and length calculation for lines
  • Create geodesic polygons such as line tool, but as a closed surface
  • Perimeter and area calculation for polygons
  • Inserting text, images, links, etc. (HTML Box)
  • Info window now possible for all elements
